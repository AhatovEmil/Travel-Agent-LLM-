# Roadmap — Travel Agent MVP

1. Backend-каркас — FastAPI, конфиг, БД (PostgreSQL / SQLite).
2. Модели — User, Trip, Artifact.
3. Авторизация — register, login, JWT.
4. CRUD поездок — создание, список, детали, удаление; изоляция по владельцу.
5. Конвейер агента — 4 фазы (Brief, Itinerary, Budget, Checklist), фон, статусы.
6. DeepSeek LLM-движок — промпты, ретраи, понятные ошибки.
7. Frontend — логин, дашборд поездок, страница поездки с фазами.
8. Docker — backend, frontend, PostgreSQL.
9. Тесты — auth, trips, пайплайн с моком LLM.
10. Документация — README, API, Architecture, Deployment.

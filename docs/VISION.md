# Project Vision — Travel Agent

## Суть продукта

**Travel Agent** — веб-приложение, в котором пользователь описывает поездку
на естественном языке, а ИИ-агент (DeepSeek) собирает готовый план путешествия:
уточнённое ТЗ, маршрут по дням, бюджет и чеклист вещей.

## Business Goal

Сократить путь «хочу куда-то съездить» → «понятный план поездки» с часов поиска
до минут. Дать черновик, который можно сразу использовать и уточнять.

## Target Audience

1. Туристы, которые планируют поездку сами, без турагентства.
2. Пары и небольшие компании, которым нужен общий план и бюджет.
3. Люди, которым лень собирать информацию из десятков вкладок.

## MVP Scope

1. Регистрация и вход (JWT).
2. Создание поездки: название + описание (город, дни, бюджет, интересы).
3. Конвейер агента из 4 фаз: Brief → Itinerary → Budget → Checklist.
4. Просмотр артефактов каждой фазы в UI.
5. Статус выполнения (polling).
6. DeepSeek как LLM-провайдер (ключ обязателен).
7. Docker Compose, автотесты, документация.

## Future Scope

- Уточняющий диалог перед генерацией.
- Карты и точки интереса.
- Live-погода и курсы валют.
- Экспорт в PDF / Google Calendar.
- Бронирование отелей и билетов (партнёры).

## Риски

| Риск | Митигация |
|---|---|
| LLM недоступен / нет ключа | Честная ошибка 503 / failed, без фейковых шаблонов |
| Галлюцинации адресов и цен | Пометка в UI: «черновик, проверяйте факты» |
| Долгая генерация | Фоновый пайплайн + polling |
